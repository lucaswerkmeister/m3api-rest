Class: IncompatibleResponseType

IncompatibleResponseType(expectedType, actualType, body)

An error representing a HTTP response with a content type incompatible with that requested. For example, if the API responded with HTML despite JSON having been requested. Note that, depending on server behavior, this condition may also be represented by a RestApiClientError with the status 406 Not Acceptable; IncompatibleResponseType is thrown by m3api-rest if the server responded with a successful status that nevertheless does not match the requested type.
